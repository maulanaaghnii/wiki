## 🛠️ Local Development

To run this wiki locally, ensure you have Python installed, then follow these steps:

1. **Clone the repository:**
   ```powershell
   git clone https://github.com/maulanaaghnii/wiki.git
   cd wiki
   ```

2. **Install dependencies:**
   ```powershell
   pip install mkdocs-material mkdocs-blog-plugin
   ```

3. **Serve the site:**
   ```powershell
   mkdocs serve
   ```
   *The wiki will be available at `http://localhost:8000`*
